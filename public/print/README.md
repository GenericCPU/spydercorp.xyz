# SpyderCorp business cards

Print-ready assets for standard US business cards.

| File | Description |
|------|-------------|
| `business-card-front.svg` / `.png` | Logo + domain + tagline |
| `business-card-back.svg` / `.png` | Contact + services |
| `business-card-preview.png` | Both sides side-by-side |

## Specs

- **Trim size:** 3.5″ × 2″ (89 × 51 mm)
- **Pixel size @ 300 DPI:** 1050 × 600 px
- **Background:** White (`#ffffff`)
- **Accent:** `#e30613`

For professional printing, upload the **SVG** or **PNG** files. If your printer requires bleed, extend the canvas by 0.125″ (38 px @ 300 DPI) on each edge and keep important content inside the safe zone.

Regenerate PNGs:

```bash
npm run print:cards
```
